III 型积分的轮换对称性

对于∫Xf dU\int_X f \ \mathrm{d}U∫X​f dU：

1、被积函数不变，积分区域不变

没有意义，因为无法简化运算

2、被积函数改变，积分区域不变

（1）两字母轮换：将 x,yx,yx,y 换为 y,xy,xy,x，积分域 XXX 不变（几何意义：积分域 XXX 关于 x=yx=yx=y 对称），而被积函数 f(x,y)f(x,y)f(x,y) 变为 f(y,x)f(y,x)f(y,x)，则

(1)∫Xf(x,y) dU=∫Xf(y,x) dU(1)\int_X f(x,y) \ \mathrm{d}U=\int_X f(y,x) \ \mathrm{d}U (1)∫X​f(x,y) dU=∫X​f(y,x) dU

(2)∫Xf(x,y) dU=12(∫Xf(x,y) dU+∫Xf(y,z) dU)(2)\int_X f(x,y) \ \mathrm{d}U = \frac{1}{2}\bigg(\int_X f(x,y) \ \mathrm{d}U+\int_X f(y,z) \ \mathrm{d}U\bigg) (2)∫X​f(x,y) dU=21​(∫X​f(x,y) dU+∫X​f(y,z) dU)

（2）三字母轮换：将 x,y,zx,y,zx,y,z 换为 y,z,xy,z,xy,z,x 积分域 XXX 不变（几何意义：积分域 XXX 绕直线 x=y=zx=y=zx=y=z 旋转120°, 积分域 XXX 不变），而被积函数 f(x,y,z)f(x,y,z)f(x,y,z) 变为 f(y,z,x)f(y,z,x)f(y,z,x),则

(1)∫Xf(x,y,z) dU=∫Xf(y,z,x) dU=∫Xf(z,x,y) dU(1)\int_X f(x,y,z) \ \mathrm{d}U=\int_X f(y,z,x) \ \mathrm{d}U=\int_X f(z,x,y) \ \mathrm{d}U (1)∫X​f(x,y,z) dU=∫X​f(y,z,x) dU=∫X​f(z,x,y) dU

(2)∫Xf(x,y,z) dU=13(∫Xf(x,y,z) dU+∫Xf(y,z,x) dU+∫Xf(z,x,y) dU)(2)\int_X f(x,y,z) \ \mathrm{d}U = \frac{1}{3}\bigg(\int_X f(x,y,z) \ \mathrm{d}U+\int_X f(y,z,x) \ \mathrm{d}U+\int_X f(z,x,y) \ \mathrm{d}U\bigg) (2)∫X​f(x,y,z) dU=31​(∫X​f(x,y,z) dU+∫X​f(y,z,x) dU+∫X​f(z,x,y) dU)

3、被积函数不变,积分区域改变

（1）两字母轮换：将 x,yx,yx,y 换为 y,xy,xy,x，积分域 XXX 改变为 X1X_1X1​ （几何意义：积分域 XXX 与 X1X_1X1​ 关于 x=yx=yx=y 对称），而被积函数 f(x,y)f(x,y)f(x,y) 不变，则

∫Xf(x,y) dU=∫X1f(x,y) dU\int_X f(x,y) \ \mathrm{d}U=\int_{X_1} f(x,y) \ \mathrm{d}U ∫X​f(x,y) dU=∫X1​​f(x,y) dU

（2）三字母轮换：将 x,y,zx,y,zx,y,z​ 换为 y,z,xy,z,xy,z,x​，积分域 XXX​ 改变为 X1X_1X1​ （几何意义： 积分域 XXX 绕直线 x=y=zx=y=zx=y=z​ 旋转120°，XXX​ 变为 X1X_1X1​​），而被积函数 f(x,y,z)f(x,y,z)f(x,y,z) 不变，则

∫Xf(x,y,z) dU=∫X1f(x,y,z) dU\int_X f(x,y,z) \ \mathrm{d}U=\int_{X_1} f(x,y,z) \ \mathrm{d}U ∫X​f(x,y,z) dU=∫X1​​f(x,y,z) dU

IIIIII 型积分的轮换对称性

1、第二类曲面积分

曲面关于变量的轮换对称性的几何意义是指曲面关于直线 x=y=zx = y = zx=y=z 对称，即将曲面绕直线
x=y=zx=y=zx=y=z 旋转 2π3\frac{2π}{3}32π​ 或 4π3\frac{4π}{3}34π​ 弧度后，仍与原曲面相重合。如平面 x+y+z=1x + y + z = 1x+y+z=1，球面 x2+y2+z2=a2x^2 + y^2 + z^2 = a^2x2+y2+z2=a2 等均属此类。如果曲面 Σ\SigmaΣ 具有轮换对称性，则曲面的方程 F(x,y,z)=0F(x, y, z) = 0F(x,y,z)=0 必有如下特征：将 F(x,y,z)F(x, y, z)F(x,y,z) 中的变量x，y，zx，y，zx，y，z 的位置轮换，不会改变函数 F(x,y,z)F(x, y, z)F(x,y,z) 的表达式。如 F(x,y,z)=x2+y2+z2−a2F(x, y, z) = x^2 + y^2 + z^2 - a^2F(x,y,z)=x2+y2+z2−a2，则 F(x,y,z)=z2+x2+y2−a2F(x, y, z) = z^2 + x^2 + y^2 - a^2F(x,y,z)=z2+x2+y2−a2，两者的形式不会发生变化。

如果将 x,y,zx,y,zx,y,z 换为 y,z,xy,z,xy,z,x 或 z,x,yz,x,yz,x,y，曲面 Σ\SigmaΣ 及其侧不变（几何意义：Σ\SigmaΣ 及其侧绕 x=y=zx=y=zx=y=z 旋转 2π3\frac{2π}{3}32π​ 或 4π3\frac{4π}{3}34π​ 弧度不变），即曲面 Σ\SigmaΣ 在各坐标平面上的投影区域相同，且配给的符号也相同，则

(1)∬Σf(x,y,z) dy dz=∬Σf(y,z,x) dz dx=∬Σf(z,x,y) dx dy(1)\iint_\Sigma f(x,y,z) \ \mathrm{d}y\,\mathrm{d}z = \iint_\Sigma f(y,z,x)\ \mathrm{d}z \, \mathrm{d}x = \iint_\Sigma f(z,x,y) \ \mathrm{d}x\,\mathrm{d}y (1)∬Σ​f(x,y,z) dydz=∬Σ​f(y,z,x) dzdx=∬Σ​f(z,x,y) dxdy

(2)∬Σf(x,y,z) dy dz=13(∬Σf(x,y,z) dy dz+∬Σf(y,z,x) dz dx+∬Σf(z,x,y) dx dy)(2)\iint_\Sigma f(x,y,z) \ \mathrm{d}y\,\mathrm{d}z = \frac{1}{3}\bigg(\iint_\Sigma f(x,y,z) \ \mathrm{d}y\,\mathrm{d}z+\iint_\Sigma f(y,z,x)\ \mathrm{d}z \, \mathrm{d}x+\iint_\Sigma f(z,x,y) \ \mathrm{d}x\,\mathrm{d}y\bigg) (2)∬Σ​f(x,y,z) dydz=31​(∬Σ​f(x,y,z) dydz+∬Σ​f(y,z,x) dzdx+∬Σ​f(z,x,y) dxdy)

2、第二类曲线积分

如果将 x,y,zx,y,zx,y,z 换为 y,z,xy,z,xy,z,x 或 z,x,yz,x,yz,x,y，空间曲线 LLL 及其方向不变（几何意义：LLL 及其方向绕 x=y=zx=y=zx=y=z 旋转 2π3\frac{2π}{3}32π​ 或 4π3\frac{4π}{3}34π​ 弧度不变），则

(1)∫Lf(x,y,z) dx=∫Lf(y,z,x) dy=∫Lf(z,x,y) dz(1)\int_L f(x,y,z) \ \mathrm{d}x = \int_L f(y,z,x) \ \mathrm{d}y = \int_L f(z,x,y) \ \mathrm{d}z (1)∫L​f(x,y,z) dx=∫L​f(y,z,x) dy=∫L​f(z,x,y) dz

(2)∫Lf(x,y,z) dx=13(∫Lf(x,y,z) dx+∫Lf(y,z,x) dy+∫Lf(z,x,y) dz)(2)\int_L f(x,y,z) \ \mathrm{d}x = \frac{1}{3}\bigg(\int_L f(x,y,z) \ \mathrm{d}x+\int_L f(y,z,x) \ \mathrm{d}y+\int_L f(z,x,y) \ \mathrm{d}z\bigg) (2)∫L​f(x,y,z) dx=31​(∫L​f(x,y,z) dx+∫L​f(y,z,x) dy+∫L​f(z,x,y) dz)
